# Compact text Editor

This is the stripped down version of the project **Text-Editor**(visit repository for details). It supports nearly all of it's features, but fetches scripts and components from the repository of **text-editor**.
This distribution supports a completely new compact look, modern and slightly classic feel.

## Main Features

- Responsive design across screen sizes  
- Clean and consistent theme
- Emojis used for buttons
- Nearly all features of original **Text-Editor**
